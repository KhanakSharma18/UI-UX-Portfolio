UX Design Figma
01 / 04

Budgetly

A complete UI/UX case study designed and prototyped in Figma from problem discovery to final solution.

In this i engineered a premium, glassmorphic dark-mode web ecosystem to eliminate user "financial fog" and manual tracking fatigue by transforming passive financial reporting into an actionable dashboard command center. Conducted targeted user research among working professionals aged 22–35 to identify key UX friction points, leading to a complete architectural overhaul that introduced an intuitive central Quick Access component. By optimizing spatial layout, simplifying text-heavy notification elements, and contextually pairing live transaction ledgers with an interactive temporal calendar view, the redesigned interface successfully reduced the average user interaction paths and clicks required to log daily expenses by 50% while accelerating overall task completion rates by 40%.

Women Safety Mobile App
03 / 04

RAKSHAK

safety mobile application featuring proactive emergency response mechanics and automated risk-detection triggers.

Designed Rakshak, an AI-powered personal safety mobile application tailored for high-stress scenarios where minimizing cognitive load is critical. The user experience centers around a persistent, high-visibility, one-tap SOS button and clean utility tiles like "Fake Call" and "Share Location" to enable immediate action. To balance proactivity with user control, the application features an "AI Guardian" settings panel that lets users customize background security automation—such as voice-activated distress triggers ("Help" or "Bachao"), shake-to-SOS mechanics, and automated late-night location risk checks. The ecosystem is tied together by an intuitive, high-scannability dashboard for quick-dial emergency helplines and an easy-to-manage "Trusted Contacts" portal, ensuring seamless real-time location sharing and instant distress broadcasting to family networks and local authorities.
